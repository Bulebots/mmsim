What is Micromouse Maze Simulator?
----------------------------------

Micromouse Maze Simulator is a micromouse search algorithm simulator. It acts
as a server, so any client can connect to it and send requests to read walls
and log the current exploration state. `Watch it in action!
<https://youtu.be/6Om6xij6LAw>`_
